Installation process

You will need the following in order to install our Application to your PC / Server

  1. Use git clone https://github.com/yaiestura/coronavirus_prediction.git to clone this repository to your computer
  2. Install pip3 on your system if not already installed by executing: 
    sudo apt-get install python3-pip
  3. Create a virtual environment by the name of venv:
    virtualenv venv
  4. Activate your virtualenv by: 
    source venv/bin/activate
  5. Execute a command to install the required packages
    6. pip3 install -r requirements.txt
  6. Open a terminal and enter
    7. python3 run.py
  7. Finally, go to localhost:5000 to display the start page of application
  8. Or you can just run a bash script:
    9. sudo bash deploy.sh

Be careful while editing the application code. If not edited properly, the design layout and architecture may break completely.
No support is provided for faulty customization.
